# Project setup
* install python 
* install pip
* install flask 
* in windows os run in cmd
 ```
  set FLASK_APP=app.py 
  flask run
 ```
* in linux os run in terminal 
```
   export FLASK_APP=app.py
   flask run
```
* go in your browser http://127.0.0.1:5000
